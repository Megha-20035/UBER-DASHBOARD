# Uber Analytics Dashboard

## Overview

This project is an interactive Power BI dashboard designed to analyze Uber business performance through multiple analytical pages including Homepage, Overview, Vehicle Analysis, and Revenue Insights. The dashboard provides a complete view of ride operations, revenue trends, vehicle performance, and customer activity using dynamic visualizations and KPI tracking.

## Project Objectives

* Monitor overall Uber business performance
* Analyze trip and booking trends
* Track revenue growth and operational metrics
* Evaluate vehicle-wise performance
* Identify peak usage periods and customer demand
* Create an interactive and user-friendly reporting experience
